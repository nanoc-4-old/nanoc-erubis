# nanoc-erubis

This provides an [Erubis](http://www.kuwata-lab.com/erubis/) filter for [nanoc](http://nanoc.ws).

## Installation

`gem install nanoc-erubis`

## Usage

```ruby
filter :erubis
```

This filter takes no options.
